# Smart-Energy-Meter-System

## Description
Real-time smart energy meter to track and analyze home power usage. 
Measures Voltage, Current, Power, Energy and logs to SD card.

## Components Required
- Arduino Uno / Mega
- PZEM-004T Energy Meter Sensor
- 16x2 LCD Display
- SD Card Module

## Features
1. Real-time monitoring of V, I, P, E
2. Data logging to energy.csv for analytics
3. LCD display for live readings

## Real-World Application
Home energy management, Smart billing, Power saving
